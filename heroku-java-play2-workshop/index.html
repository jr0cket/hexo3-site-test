Heroku Workshop - Java & Play framework

This workshop will give you experience deploying and managing an application on Heroku, a cloud application platform. Before you start the workshop there will be a presentation on Heroku, describing the service and giving you a basic idea of how to deploy scalable applications.

A great way to understand the value of Heroku is to use it in your projects. In this workshop we try and simulate the project experience as much as we can, although there are limits. The workshop you will guide you through all the steps to build a relatively simple Java application using the Play framework.

The workshop is arranged in the following sections:

  • Chapter 0: Setting up your environment
  • Chapter 1: Getting started with your app
  • Chapter 2: Manage your project changes with Git
  • Chapter 3: Deploy your application on Heroku
  • Chapter 4: Developer workflow for Git and Heroku
  • Chapter 5: Setting up an IDE for Play framework (optional)
  • Chapter 6: Developing the app: Adding Routes
  • Chapter 7: Developing the app: The Task model
  • Chapter 8: Developing the app: Persisting the data
  • Chapter 9: Using Heroku Postgres database
  • Chapter 10: Releases and Rollbacks
  • Chapter 11: Working with Heroku Logs
  • Chapter 12: Searching logs with Papertrail add-on (optional)
  • Chapter 13: Adding a Staging Environment
  • Chapter 14: Managing Multiple Git Repositories
  • Chapter 15: Collaborating with Github
  • Chapter 16: Monitoring Heroku apps
  • Chapter 17: Performance Monitoring with New Relic (optional)
  • Appendix A: Additional Resources
